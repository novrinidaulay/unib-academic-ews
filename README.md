# 📊 UNIB Smart Academic Predictor (Early Warning System)

Proyek ini merupakan purwarupa (*proof of concept*) dari sistem penunjang keputusan berbasis *Data-Driven Governance*. Aplikasi ini dirancang khusus sebagai gagasan inovasi layanan akademik terintegrasi untuk posisi **Kepala Biro Akademik dan Kemahasiswaan, Universitas Bengkulu**.

## 🎯 Latar Belakang Inovasi
Sejalan dengan tugas dan fungsi Biro Akademik dan Kemahasiswaan dalam mengevaluasi pendidikan dan mengelola sarana akademik, volume data mahasiswa yang masif memerlukan pendekatan berbasis **Sains Data (Data Science)** dan **Analisis Prediktif**. Sistem ini bertujuan untuk mendeteksi secara dini mahasiswa yang berisiko terlambat lulus atau *drop-out* agar Biro Akademik dapat memberikan intervensi kebijakan yang cepat, tepat sasaran, dan berbasis data.

## ✨ Fitur Utama
* **Deteksi Dini (Early Warning):** Menggunakan algoritma *Machine Learning* (Random Forest Classifier) untuk memprediksi probabilitas kelulusan mahasiswa tepat waktu.
* **Dasbor Interaktif:** Antarmuka ramah pengguna bagi staf akademik untuk memasukkan indikator mahasiswa (IPK, Kehadiran, SKS Gagal) secara *real-time*.
* **Visualisasi Kebijakan (Feature Importance):** Pemetaan faktor-faktor paling krusial yang memengaruhi masa studi mahasiswa sebagai landasan pengambilan keputusan strategis.

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Python
* **Machine Learning:** Scikit-Learn
* **Visualisasi & Pengolahan Data:** Pandas, NumPy, Matplotlib, Seaborn
* **Deployment & Antarmuka:** Streamlit
